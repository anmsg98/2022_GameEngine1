#개발 내용 및 문제점
- 스킬을 전체적으로 관리하는 싱글톤 기반 SkillManager 구현
- 스킬 슬롯 기능 구현 및 스킬 사용시 상태 UI 연계 및 쿨타임 기능 구현
- 전체적인 Player 코드 수정(Megan만 애니메이션 설정을 변경했기 때문에 다른 캐릭터들은 아직 안됨)
- 각 SkillPrefab의 isUsable=true, skillLevel=0, currentTime=0을 Inspector에서 설정해주고 실행해야함, 왜 플레이를 마쳤을 때 다시 원래 값으로 돌아오지 않고 마지막 값이 저장되는지 모르겠음
- 파티클 생성 위치 버그 문제 해결 - 플레이어의 RigidBody 컴포넌트를 제거하니 정상동작 => 정확한 이유는 모르겠음
- 오브젝트 풀링을 이용하여 몬스터 생성시, 좌표값이 이상한 버그 해결 - NavMeshAgent 컴포넌트 문제임(Instantiate(PREFAB)과 같이 오버로딩된 함수를 사용하고 position과 rotation을 설정하게 되면, 최초 생성은 0, 0, 0이기 때문에 NavMesh에 붙어버려서 y값을 변경할 수 없게됨) => 착지시에 해당 컴포넌트를 켜고 사망시 다시 끄는 식으로 변경
